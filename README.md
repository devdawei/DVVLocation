

DVVLocation
===========
对百度地图地理编码功能的封装。

使用方式
-------
在`Podfile`里添加此行内容：
```
pod 'DVVLocation', :git => 'https://github.com/devdawei/DVVLocation.git', :tag => 'v1.0.1'
```

然后在`Terminal`中`cd`到工程目录，执行如下命令：
```
pod install
```

因为用到了DVVAlertView，所以还需要您导入DVVAlertView组件。[点击前往](https://github.com/devdawei/DVVAlertView)
